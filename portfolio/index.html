Alpha Project Notes

First Time for Everything

This is the first time I did anything serious with HTML/CSS. I've played with it enough but I'd never built a major project using it. I built the page with the first real utilization of JavaScript. I'd done some very basic JS-based graphics, but it was interesting being in this situation. After banging my head into the wall enough times figuring out the path to take; and using a lot of salty language when talking to something that (I hope) can't hear me, I finally finished. I had the distinct feeling that I was insane while I screamed "It's Alive!" Actually, I did that silently. My wife and mother-in-law might have thought I'd lost my mind if I really did that, though my toddler-age son would probably find it very funny. There's a chance that they'd be right, but I don't want to give them any more ideas than neccessary.

This spot can be filled buy any number of projects through the Udacity classes, and I expect it to be filled in such a way with the next set of projects after this one

Dylan's Page (Project Bravo)

Dylan's Website

This is a project that is still ongoing. It's almost exclusively based around photos of my son. What parent doesn't have dozens of picturs of their child? I decided that it would be a good way to get more practice, without a preplanned design and without client or deadline. As you can imagine, since it's mostly practice it's still under construction. I spend what free time I have working on it. Free time is wedged between the work I do on programming courses and the time I spend with my son. The largest reason I continue on this project is that I can put any new knowledge I gain to use in it, and any failure is minor and doesn't have a major impact on anything else. I have a design completed and I'm currently building the guts of the site now.
